# 快速开始

## 四步开始使用

### 第一步：配置模型

在「设置」页面填写 API Base URL 和 API Key，点击「获取」自动拉取模型列表后选择，也可手动输入模型 ID。

### 第二步：选择识别模式

在「设置 → 模型配置 → 识别模式」中选择：

- **视频模式**（默认）：上传视频片段，适合豆包、Gemini 等支持视频的模型
- **图片模式**：逐帧截图分析，适合 GPT-4o、Qwen-VL 等视觉模型

### 第三步：开始录制

点击左侧侧边栏底部的「开始录制」按钮。程序将以 1 FPS 截取屏幕，每 15 秒打包为一个视频片段保存到 `recordings/` 目录。

### 第四步：查看时间线

切换到「时间线」视图，用日期导航浏览每天的活动，点击卡片查看详细描述。

## 注意事项

- 首次使用需先配置 API Key，否则分析功能不可用
- 录制期间 LLM 分析在后台自动运行，无需手动触发
- 可在「仪表盘」查看 Token 用量和分析进度
