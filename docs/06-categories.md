# 活动分类

每个活动卡片由 LLM 自动归入以下四种类型之一。

## 分类定义

- **工作** — 编程、写作、会议、邮件等职业任务
- **学习** — 课程、教程、文档阅读、技术研究
- **娱乐** — 视频、游戏、社交媒体浏览
- **其他** — 不属于以上分类的活动

## 在时间线中查看

时间线视图中，每个活动卡片会显示对应的分类标签，并以不同颜色区分，方便快速识别一天的时间分配。

## 关于分类准确性

LLM 根据屏幕内容自动判断分类，对于边界模糊的活动（如技术文档阅读算「工作」还是「学习」）可能因上下文而有所不同，这是预期行为。
