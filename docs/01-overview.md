# Workday 简介

Workday 是一款 AI 驱动的工作时间追踪工具，自动录制屏幕并通过 LLM 生成每日活动时间线。

## 核心特性

- 自动以 1 FPS 录制屏幕，每 15 秒打包为一个视频片段
- 两阶段 LLM 分析流水线：视频转录 + 活动卡片生成
- 支持任何兼容 OpenAI API 格式的服务商
- 所有数据本地存储，不上传云端（仅 LLM 调用时传输屏幕内容）
- 跨平台支持（macOS / Windows / Linux）

## 基本工作流程

录制屏幕 → 15 秒打包视频片段 → 自动提交后台分析 → LLM 生成活动时间线 → 在「时间线」视图查看
