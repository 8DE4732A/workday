# Token 用量

## 查看用量

在「仪表盘」页面可查看每次 AI 分析消耗的 Token 数量，支持按日期过滤。

## 计费结构

每个批次的分析包含两次 LLM 调用：

1. **转录调用**（第一阶段）：输入视频/图片 + 少量文本提示
2. **卡片生成调用**（第二阶段）：输入观察记录文本，纯文本处理

视觉输入（视频/图片）通常比纯文本消耗更多 Token，具体取决于模型计费规则。

## 节省 Token 的建议

- 启用静止帧过滤（视频模式的 `static_frame_ratio` 或图片模式的 `static_diff_threshold`）
- 录制期间避免长时间停留在静止画面
- 适当调大批次时长，减少分析批次数量

## 数据持久化

Token 用量记录存储在本地 `workday.db` 的 `token_usage` 表中，不会丢失。
